# MyNewsApp

## 中文介紹

MyNewsApp 是一個使用 Android Studio 開發的科技新聞瀏覽 App。
使用者可以點擊不同的新聞主題按鈕，例如：

* 航太
* 人工智慧
* 氣象科技
* 生物醫學
* 計算

即可查看對應分類的科技新聞內容。

### 功能特色

* 科技新聞分類瀏覽
* 簡潔直觀的操作介面
* 點擊分類即可切換新聞內容
* 使用 Android 原生開發

### 開發工具

* Android Studio
* Java
* XML Layout

### 專案名稱

`MyNewsApp`

# MyNewsApp

## English Introduction

MyNewsApp is a technology news browsing app developed with Android Studio.
Users can tap different topic buttons such as:

* Aerospace
* Artificial Intelligence
* Weather Technology
* Biomedical Science
* Computing

to view related technology news articles.

### Features

* Browse technology news by category
* Simple and clean user interface
* Tap categories to switch news content
* Built with native Android development

### Built With

* Android Studio
* Java
* XML Layout

### Project Name

`MyNewsApp`
